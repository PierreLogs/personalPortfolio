Portfolio web personal responsive con modo oscuro/claro, diseñado para mostrar proyectos, experiencia y formación como desarrollador.

## Características

- Modo oscuro/claro con persistencia en `localStorage`
- Diseño responsive (mobile-first)
- Navegación smooth-scroll
- Botón de descarga de CV
- Enlace directo de contacto por email

## Stack

- HTML5 semántico + BEM
- CSS3 con custom properties (variables para temas)
- JavaScript vanilla (DOM, localStorage, eventos)
- Remixicon + Google Fonts (Poppins, Sora)

## Demo en vivo

👉 [pierrelogs.github.io/personalPortfolio](https://pierrelogs.github.io/personalPortfolio/)

## Uso

Abre `index.html` en cualquier navegador moderno. No requiere build ni servidor.
